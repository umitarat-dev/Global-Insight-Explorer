## ✨ Overview

**Flag App** is a beginner-friendly JavaScript project that fetches country data from a public API and displays flags along with basic country information.

The project focuses on working with **asynchronous data**, **Fetch API**, and rendering dynamic content on the page using vanilla JavaScript.

## 🚀 Features

- 🌍 Fetch country data using **Fetch API**
- 🏳️ Display country flags dynamically
- ⚡ Asynchronous data handling with `async / await`
- 📱 Responsive UI with Bootstrap 5
- 🎯 Clean and simple user interface

## 🛠️ Built With

- **HTML5**
- **CSS3**
- **JavaScript (Vanilla JS)**
- **Fetch API**
- **Async / Await**
- **Bootstrap 5**
